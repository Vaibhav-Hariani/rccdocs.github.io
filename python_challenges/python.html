Differentiation

Main Quest:

  • Create a function that finds the velocity at a particular point in time of an object's trajectory, given a particular function that describes distance traveled over time.
    • The function should pass in TWO arguments: the time of interest (tau) and the delta-t (dt)
    • Your code should return a single value, the velocity/derivative
    • For this activity, set the time of interest, tau, to 0 seconds, and delta-t, dt, to 0.5 seconds.

Method:

  • Using the function: \(x = 3t^2 + 5t + 7\)

    1. Find the distance, x, traveled at time tau by plugging in tau into the function.

    2. Find the distance traveled at time tau + dt by plugging tau + dt into the function

    3. Find the slope of the line that spans the two points. That is your velocity at time tau.

    4. Call the function again, but this time with delta-t = 0.1. Do it again with delta-t = 0.001s, 0.00001s. Do you notice anything?

Integration

Main Quest:

  • Create a function that finds the distance traveled at a particular point in time of an object's trajectory, given a particular function for the velocity.
    • The function should pass in THREE arguments: the start time of interest (tau1), the end time of interest (tau2), and the delta-t (dt)
    • Your code should return a single value, the distance/interal
    • For this activity, set the time interval of interest to be from 0 to 3 seconds, and delta-t, dt, to 0.5 seconds.

Method:

  • Using the function: \(v = 6t + 5\)

    1. Determine how many rectangles will be calculated under the curve. Use this number to define the upper limit of the range for the iterator in your loop. (If you are confident, try writing an expression for how many rectangles are under the curve; if delta-t changes, the number )

    2. Use the Right-Hand Riemann sum:
      1. The first area calculation will be using the minimum value of the range plus delta-t; in our case, it would be 0 + 0.5. Calculate the area of the rectangle, base * height, where the base is delta-t, and the height is v(0+0.5).
      2. Calculate the areas of the other rectangles, using the heights of v(0+2*0.5), v(0+3*0.5, etc.)
      3. Add all the calculated areas up to find the approximation of the area under the curve.

    3. Output the integral; this is the amount of distance covered from time t = 0 to t = 3.

    4. Change delta-t to 0.1. Change it to 0.001. What do you notice?

    5. Notice that, as opposed to the differentiation quests, the integral isn't AT a particular point in time, but rather BETWEEN two points in time. Try changing the time intervals to 0s to 1s, and 1s to 3s. See if it adds up to your original estimation.

Side Quest 1:

  • Create a function that stores the current total area in an array as the loop iterates through the number of rectangles.
    • The function should pass in THREE arguments: the time interval, describes by tau1 and tau2, and the delta-t (dt)
    • Your code should iterate and find the area of the first rectangle, the first two rectangles, the first three rectangles, etc. and stores the growing total area as elements in an array.
    • Calculate the distances from t = 0s to t = 3s using a delta-t of 0.001s (basically your array should have 3000 elements)
    • In other words, the array should be a running total of the distance traveled by the robot.

Method:

  • Using the function: \(x = 6t + 5\)

    1. Take the right-hand Riemann sum of each rectangle.

    2. Add it to the total area.

    3. Append the current total area to the array.

    4. Repeat until the end of the time interval.
