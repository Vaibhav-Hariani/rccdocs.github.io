Differentiation

★ Main Quest:

  • Create a python function that calculates a derivative
    • The function should pass in TWO arguments: the time of interest \(tau\) and the delta-t, \(\Delta t\)
    • Your code should return a single value, the velocity
    • For this activity, set \(tau\) to 0 seconds and \(\Delta t\) to 0.5 seconds

Method:

  • Using the polynomial: \(x = 3t^2 + 5t + 7\)

  • The function should:

    1. Find the distance traveled at time \(tau\)

    2. Find the distance traveled at time \((tau + \Delta t)\)

    3. Find the slope of the line that spans the two points

    4. Return the estimated velocity

  • Call the function again, but this time with \(\Delta t\) = 0.1
  • Do it again with \(\Delta t\) = 0.001 and \(\Delta t\) = 0.00001
  • Do you notice anything?

★★ Side Quest 1:

    • The function should pass in THREE arguments: the start_time, the end_time, and the delta_t, \(\Delta t\)
    • Calculate the velocities from t = 0s to t = 3s using a \(\Delta t\) of 0.001s (hint: your array should have 3000 elements)

Method:

  • Using the function: \(x = 3t^2 + 5t + 7\)

    1. Find the distance, \(x\), traveled at the start time and one \(\Delta t\) from the start time. Calculate the velocity.

    2. Do the same with \(2\Delta t\), \(3\Delta t\), and so forth, store all the velocities in the array.

    3. Return the array of velocities

Integration

★ Main Quest:

  • Create a Python function that finds the distance traveled given the polynomial describing velocity.
    • The function should pass in THREE arguments: the start_time, the end_time, and
    • Return a single value, the distance traveled
    • For this activity, set the time interval to be from 0 to 3 seconds, and \(\Delta t\) to 0.5 seconds.

Method:

  • Using the polynomial: \(v = 6t + 5\)

    1. Determine how many \(\Delta t\) are in the time interval

    2. Use the Right-Hand Riemann sum method

      • Hint: In the loop, calculate the area of each rectangle
      • Sum all of the areas to estimate the total distance traveled

    3. Return your estimation

    4. Change \(\Delta t\) to 0.1. Change it to 0.001. What do you observe?

    5. Notice that as opposed to the differentiation quests, the integral isn't AT a particular point in time, but rather BETWEEN two points in time. Try changing the time intervals to 0s to 1s, and 1s to 3s. See if it adds up to your original estimation.

★★ Side Quest 1:

  • Create a function that will store the accumulated area at each timestep in an array

Method:

  • Using the function: \(x = 6t + 5\)

    1. Take the right-hand Riemann sum of each rectangle.

    2. Add it to the total area.

    3. Append the current total area to the array.

    4. Repeat until the end of the time interval.
